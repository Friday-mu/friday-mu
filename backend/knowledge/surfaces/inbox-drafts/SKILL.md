# Inbox Drafts

## Cadence

- Pre-arrival (48h): Full address, check-in window (3-8 PM), access codes, WiFi, WhatsApp contact, weather, tourist tax, nearest pharmacy/supermarket
- Welcome day: Personal welcome, dinner recs, sunset spot, tomorrow's activities, emergency contacts
- Mid-stay (day 3+): Satisfaction check, weather forecast, local events, mid-stay cleaning offer (5+ nights)
- Pre-departure: Checkout time (10 AM), key return, simple checklist (max 5-6 items). Never ask guests to clean.
- Post-checkout (24h): Thank you, warm sign-off, FRIDAY10 code for 10% off direct booking

## Assembly Hints

- Open with the substance of the guest's need. Skip greeting filler.
- Match length to substance: short confirmation → short reply; multi-part question → cover each part.
- Default to prose paragraphs. Bullets only for procedural stepwise info.
- Deliver requested facts inline (codes, times, addresses). No deferral wrapping.
- Use commas, periods, parentheses, colons. No em-dashes.
- Close hospitality comms with warmth: "Safe travels", "Pleasure hosting".
