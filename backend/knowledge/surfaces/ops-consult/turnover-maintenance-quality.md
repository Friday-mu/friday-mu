# Turnover, Maintenance, And Quality Knowledge

## STR Turnover Principles

- Cleanliness and reset quality are high-impact review drivers. Small visible failures can create guest distrust.
- Use detailed room-by-room requirements so substitute cleaners can perform consistently.
- Keep supplies on site where possible; field staff should not need to leave a turnover to buy missing basics.
- Post-clean inspection is a quality gate, not a full reclean.
- Use photos and comments for exceptions, damage, missing items, and maintenance risk.

## Standard Clean Requirements Skeleton

Property-specific requirements should override this skeleton:

- Beds and linens reset, stains/damage checked.
- Bathrooms sanitized: toilet, shower, sink, mirrors, drains, hair, soap residue.
- Kitchen cleaned: counters, sink, appliances, fridge check, dishes, trash.
- Floors vacuumed/mopped including corners and under visible furniture.
- Dust and fingerprints checked on common touchpoints.
- Trash removed and bins reset.
- Outdoor/balcony/garage/pool-adjacent areas checked if property has them.
- Welcome/SRL items restocked or verified.
- Maintenance issues reported before leaving.
- Final photos if task requires evidence.

## Post-Clean Inspection

Post-clean inspection should verify:

- Clean is guest-ready.
- SRL/welcome items are placed or missing items are captured.
- AC, Wi-Fi, hot water, lights, lockbox/access, obvious plumbing/electrical issues are checked.
- Visual appearance and smell are acceptable.
- Any urgent correction becomes a task immediately.

## Arrival Inspection

Use for properties empty more than two days before arrival. Verify access, utilities, visible cleanliness, smell, pests, pool clarity if relevant, AC, Wi-Fi, hot water, lockbox, linens/welcome pack, and any guest-specific requests.

## Preventative Maintenance

Monthly preventative maintenance should cover:

- AC filter cleaning/check.
- Plumbing leaks, drainage, water pressure, hot water.
- Electrical safety, lights, switches, extension-cord misuse.
- Doors, windows, locks, lockbox.
- Internet/router status.
- Appliance condition and noises.
- Exterior, balcony, pool/garden/provider issues if applicable.
- Safety issues and trip hazards.

Use Bryan first. Use external vendors for AC servicing until Bryan is trained.

## Maintenance Triage

Classify issue:

- Safety/hygiene/access/blocking guest comfort: urgent/high.
- Guest in house and core amenity broken: high/urgent depending impact.
- Arrival within 24h and issue affects readiness: high.
- Cosmetic/no current guest impact: low/medium.
- Unknown complexity: create assessment task for Bryan before scheduling full repair.

## Learning From Duration Deltas

Do not blindly learn from every start/stop time. If actual duration is much longer or shorter than estimated, first ask whether:

- task was left running by mistake
- staff paused but did not mark pause
- task included extra hidden work
- travel was included accidentally
- task scope was larger than title

Only update duration rules after human verification or repeated consistent evidence.
